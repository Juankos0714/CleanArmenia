Glosario HTML

    Estrutura Basica del HTML

  1. <!DOCTYPE html>:Define el tipo de documento como HTML5.
  2. <html lang="en">: Elemento raiz del html, define el lenguaje en ingles
  3. <head>: Contiene metadatos sobre el documento
  4. <body>: Contiene el cuerpo del documento

    Metadatos

  1. <meta charset="UTF-8">: Define como se deben interpretar los caracteres ingresados en el html
  2. <name="viewport" content="width=device-width, initial-scale=1.0">: Define el resposive design, como la escalda de la pagina y el ancho de dispositivo
  3. <title>: Define el titulo del documento
  4. <link>: Define la relación entre el documento actual y un recurso externo.
  5. <style>: Define información de estilo para el documento, aqui se enlaza con el CSS.
  6. <base>: Especifica la URL base para todas las URLs relativas en el documento.

    Contenido de texto

  1. <h1> a <h6>: Define encabezados de diferentes niveles.
  2. <p>: Define un párrafo.
  3. <br>: Define un salto de linea
  4. <hr>: Define un cambio temático en el contenido.
  5. <pre>: Define un preformateado en el contenido

    Formato de texto

  1. <strong>: Define un texto importante con negrilla
  2. <em>: Define un pedazo de texto enfatizado.
  3. <br>: Define un salto de linea
  4. <hr>: Define un cambio temático en el contenido.
  5. <pre>: Define un preformateado en el contenido

    Citas

  1. <blockquote>: Define una seccion citado en otra fuente
  2. <q>: Define una cita corta en linea.
  3. `` Define el titulo de una obra
  4. <abbr>: Define una abreviatura o acrónimo.
  5. <pre>: Define un preformateado en el contenido

    Enlaces

  1. <a>: Define un hipervinculo

    Imagénes y multimedia

  1. <img>: Define una seccion citado en otra fuente
  2. <figure>: Especifica contenido autónomo, como ilustraciones, diagramas, fotos, etc.
  3. <figcaption>: Define una leyenda para un elemento <figure>
  4. <audio>: Define contenido de sonido.
  5. <video>: Define contenido de video.
  6. <source>: Define múltiples recursos de medios para elementos <audio> y <video>
  7. <track>: Define pistas de texto para elementos <audio> y <video>

    Listas

  1. <ul>: Define una lista desordenada.
  2. <ol>: Define una lista ordenada.
  3. <li>: Define un elemento de lista.
  4. <dl>: Define una lista de definiciones.
  5. <dt>: Define un término en una lista de definiciones.
  6. <dd>: Define la descripción de un término en una lista de definiciones.

    Tablas

  1. <table>: Define una tabla.
  2. <caption>: Define el título de una tabla.
  3. <thead>: Agrupa el contenido del encabezado en una tabla.
  4. <tbody>: Agrupa el contenido del cuerpo en una tabla.
  5. <tfoot>: Agrupa el contenido del pie en una tabla.
  6. <tr>: Define una fila en una tabla.
  7. <th>: Define una celda de encabezado en una tabla.
  8. <td>: Define una celda en una tabla.
  9. <col>: Especifica propiedades de columna para cada columna dentro de un elemento <colgroup>
  10. <colgroup>: Especifica un grupo de una o más columnas en una tabla.

    Formularios

  1. <form>: Define un formulario HTML.
  2. <input>: Define un campo de entrada.
  3. <textarea>: Define un control de entrada de texto multilínea.
  4. <button>: Define un botón clickeable.
  5. <select>: Define una lista desplegable.
  6. <optgroup>: Define un grupo de opciones relacionadas en una lista desplegable.
  7. <option>: Define una opción en una lista desplegable.
  8. <label>: Define una etiqueta para un elemento <input>
  9. <fieldset>: Agrupa elementos relacionados en un formulario.
  10. <legend>: Define un título para un elemento <fieldset>
  11. <datalist>: Especifica una lista de opciones predefinidas para controles de entrada.
  12. <output>: Define el resultado de un cálculo.

    Scripting

  1. <script>: Define un script del lado del cliente.
  2. <noscript>: Define contenido alternativo para usuarios que tienen los scripts desactivados.
  3. <canvas>: Se utiliza para dibujar gráficos mediante scripts.

    Elementos semánticos

  1. <article>: Define contenido independiente y autónomo.
  2. <aside>: Define contenido aparte del contenido de la página.
  3. <details>: Define detalles adicionales que el usuario puede ver u ocultar.
  4. <summary>: Define un encabezado visible para un elemento <details>
  5. <header>: Define un encabezado para un documento o sección.
  6. <footer>: Define un pie de página para un documento o sección.
  7. <main>: Especifica el contenido principal de un documento.
  8. <nav>: Define un conjunto de enlaces de navegación.
  9. <section>: Define una sección en un documento.

    Elementos embebidos

  1. <embed>:Define un contenedor para una aplicación externa.
  2. <object>: Define un objeto embebido.
  3. <param>: Define un parámetro para un objeto.

    Elementos interactivos

  1. <dialog>:Define una caja de diálogo o ventana.
  2. <menu>: Define una lista de comandos.
  3. <menuitem>: Define un elemento de comando que el usuario puede invocar desde un menú emergente.

    Otros elementos

  1. <bdi>: Aísla una parte del texto que puede tener un formato bidireccional diferente del texto circundante.
  2. <bdo>: Anula la dirección del texto actual.
  3. <data>: Vincula el contenido dado con una traducción legible por máquina.
  4. <time>: Define una fecha/hora.
  5. <var>: Define una variable.
  6. <wbr>: Define un posible salto de línea.
